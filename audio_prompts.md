# Audio Prompts for Letter Sounds

Generate these audio clips using ElevenLabs, PlayHT, or similar text-to-speech tools.
Save as MP3 files in `assets/sounds/letter_X.mp3` (lowercase letter).

---

## Voice Styles

### Sesame Street Voices:
- **Elmo** - High-pitched, enthusiastic, giggly
- **Cookie Monster** - Deep, grumbly, excited about everything
- **Big Bird** - Sweet, innocent, slightly naive
- **Grover** - Over-dramatic, superhero-like

### James Bond Voice:
- **Bond** - Suave, sophisticated British accent, confident

## Recommended Tools

1. **ElevenLabs** (elevenlabs.io) - Best quality, many voice clones
2. **PlayHT** (play.ht) - Good Sesame Street style voices
3. **Resemble.AI** - Custom voice creation
4. **Murf.AI** - Various character voices

## File Naming

Save files as:
- `assets/sounds/letter_a.mp3`
- `assets/sounds/letter_b.mp3`
- ... through `letter_z.mp3`

The app will automatically fallback to browser speech if any audio file is missing.
